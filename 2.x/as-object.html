# Как объект

# Предоставляемый метод

Перечисляет все методы, предоставляемые трейтом.

# make

Разрешает действие из контейнера.

MyAction::make();

// Эквивалентно:
app(MyAction::class);

# run

Разрешает и выполняет действие.

MyAction::run($someArguments);

// Эквивалентно:
MyAction::make()->handle($someArguments);
